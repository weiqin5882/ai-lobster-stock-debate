## 🎯 核心看点

- **实盘数据驱动** - 每条发言必须引用真实赔率/交易量/IV等数据
- **专业投研互撕** - 五只龙虾分别代表：基本面/技术分析/情绪周期/内幕信息/期权杠杆
- **Polymarket集成** - 每日自动抓取前20热门预测市场
- **脏话+高认知** - 用「Q3现金流暴跌28%」这种数据骂人
- **短视频神素材** - 金句密度极高，剪辑效果炸裂

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置API Key

```bash
# 复制配置文件
cp .env.example .env

# 编辑.env填入你的Deepseek API Key
# 获取地址：https://platform.deepseek.com/
```

### 3. 运行Web界面

```bash
# 启动Web服务器
python app.py

# 访问 http://localhost:3000
```

### 4. (可选) 启动定时更新

```bash
# 每天上午9点自动更新Polymarket数据
python scheduler.py
```

## 📊 Polymarket数据

### 自动更新

- **定时任务**：每天上午9点自动抓取前20热门市场
- **手动刷新**：Web界面点击"🔄 刷新"按钮
- **数据保存**：保存到 `data/hot_markets.json`

### 数据字段

每个市场包含：
- `question` - 预测问题
- `outcomes` - 结果选项及概率（如：Yes 65% / No 35%）
- `volume` - 交易量（美元）
- `liquidity` - 流动性
- `end_date` - 结算日期

### 示例市场

| 问题 | 赔率 | 交易量 |
|------|------|--------|
| 川普2024当选？ | Yes 55% / No 45% | $12M |
| 美联储3月降息？ | Yes 30% / No 70% | $8M |
| BTC突破10万？ | Yes 25% / No 75% | $5M |
